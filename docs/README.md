# PrintBridge

> 连接数字媒体与印刷工程的桥梁

## 应用简介

PrintBridge 是一款专为数字媒体技术和印刷工程领域设计的跨平台桌面应用，旨在帮助学生和从业者理解从屏幕设计到印刷输出的完整工作流程。

### 核心价值

- **跨媒介一致性**：确保设计在数字显示和印刷输出保持一致的色彩效果
- **教学辅助**：通过交互式演示和案例分析帮助学习色彩管理知识
- **印刷预检**：在印刷前自动检测潜在问题，减少打印浪费

### 目标用户

- 数字媒体技术专业学生
- 印刷工程专业学生
- 平面设计师
- 印刷从业者

## 功能模块详解

### 1. 色彩管理模块（Color Lab）

色彩管理是 PrintBridge 的核心功能之一，帮助用户理解和处理屏幕色与印刷色的差异。

#### 主要功能

| 功能 | 说明 |
|------|------|
| **ICC 配置选择** | 支持 sRGB、Adobe RGB、FOGRA39、FOGRA29、Japan Color 2001 等常用色彩配置 |
| **RGB/CMYK 转换** | 实时转换并显示两种色彩空间的数值差异 |
| **色差计算（ΔE）** | 使用 CIE76 标准计算颜色差异，ΔE < 3 表示"在色域内" |
| **软打样预览** | 模拟设计稿在不同色彩配置下的印刷效果 |
| **色彩警告** | 高亮标记超出可印刷色彩范围的区域 |

#### 使用方法

1. 点击「选择文件」导入 RGB 图像
2. 从 ICC 配置下拉菜单选择目标色彩配置
3. 点击「分析」按钮获取色彩分析结果
4. 开启/关闭软打样预览对比显示效果

#### 预期效果

- 显示图像中心点的 RGB 和 CMYK 数值
- 计算并显示色差 ΔE 值
- 以色块形式展示原始颜色和转换后的模拟印刷效果

---

### 2. 跨媒介预览模块（Cross Preview）

通过直观的可视化对比，帮助用户理解同一设计在不同媒介下的表现差异。

#### 主要功能

| 功能 | 说明 |
|------|------|
| **并排对比** | 左右两侧分别显示数字效果和模拟印刷效果 |
| **叠加对比** | 将两种效果叠加，通过透明度控制观察差异 |
| **纸张材质模拟** | 支持铜版纸、哑光纸、新闻纸三种材质效果 |
| **光源模拟** | 支持 D50、D65、荧光灯三种标准光源 |
| **观察距离预设** | 模拟不同观察距离下的视觉效果 |
| **分辨率调整** | 72-600 DPI 可调节，模拟不同印刷精度 |

#### 使用方法

1. 在色彩管理模块导入图像后自动同步到本模块
2. 选择「并排对比」或「叠加对比」标签页
3. 在右侧面板调整观察条件参数
4. 点击「刷新预览」查看更新后的效果

#### 预期效果

- **并排对比**：显示两个并排的图像面板，带分隔线滑块可调整比例
- **叠加对比**：两张图像叠加显示，通过透明度滑块控制上层层级
- 调整纸张材质时，图像饱和度和对比度会相应变化

---

### 3. 智能印刷适配模块（Print Adapter）

自动检测设计稿中可能存在的印刷问题，并提供修正建议和完整工作流。

#### 主要功能

| 功能 | 说明 |
|------|------|
| **分辨率检测** | 检查图像分辨率是否满足 300 DPI 的印刷要求 |
| **色彩模式检查** | 检测图像是否为 CMYK 色彩模式 |
| **出血检测** | 检查图像边缘是否有足够的出血区域（3mm） |
| **色域检测** | 分析图像中超出 CMYK 可印刷范围的鲜艳颜色 |
| **智能修正** | 提供一键修正建议或引导手动修正 |
| **工作流向导** | 四步流程：检测→修正→预览→导出 |

#### 使用方法

1. 在色彩管理模块导入图像后切换到本模块
2. 点击「运行印刷检测」开始分析
3. 查看检测结果和总分（0-100 分）
4. 根据问题列表中的建议进行修正
5. 按照工作流向导完成后续步骤

#### 检测结果解读

| 分数 | 状态 | 建议 |
|------|------|------|
| 80-100 | 优秀 | 可以直接用于印刷 |
| 60-79 | 良好 | 有少量警告，建议修正后印刷 |
| 40-59 | 需修正 | 存在错误，必须修正后才能印刷 |
| 0-39 | 不合格 | 存在严重问题，需要重新设计 |

### 4. 学习资源库模块（Knowledge Hub）

丰富的教学资源，帮助用户深入理解数字媒体与印刷工程的交叉知识。

#### 主要功能

| 功能 | 说明 |
|------|------|
| **案例库** | 展示真实跨媒介设计案例，包含原稿、显示效果、印刷成品 |
| **交互演示** | 可调整参数的实时色彩模拟演示 |
| **自测题库** | 选择题形式的知识测验 |

#### 案例库

目前提供以下案例：

- **品牌视觉跨媒介一致性**：分析品牌色从数字到印刷的保持方法
- **包装设计印刷适配**：解析分辨率、出血、色域等关键控制点

#### 交互演示

- **饱和度调节**：0%-150% 范围，观察对 CMYK 转换的影响
- **对比度调节**：50%-150% 范围，观察对印刷效果的影响
- 实时显示 RGB 和对应的 CMYK 数值

#### 自测题库

涵盖以下知识点：

- 印刷前检查项目
- CMYK 色彩空间理解
- 标准印刷分辨率要求
- 出血的作用和设置

## 使用教程

### 首次使用指南

#### 步骤 1：启动应用

```bash
cd printbridge
npm run dev
```

应用启动后，会显示主界面，左侧为模块导航菜单。

#### 步骤 2：导入图像

1. 在「色彩管理」模块点击「选择文件」
2. 选择 PNG、JPG、TIFF 等格式的图像文件
3. 等待图像加载完成

#### 步骤 3：进行色彩分析

1. 选择目标 ICC 配置文件
2. 点击「分析」按钮
3. 查看色彩转换结果和色差

#### 步骤 4：跨媒介预览

1. 切换到「跨媒介预览」模块
2. 图像已自动同步
3. 调整纸张材质、光源等参数
4. 观察预览效果变化

#### 步骤 5：印刷预检

1. 切换到「智能印刷适配」模块
2. 点击「运行印刷检测」
3. 根据检测结果进行修正
4. 按照工作流向导完成印刷准备

---

### 典型工作流程

```
导入图像 → 色彩分析 → 选择 ICC 配置 → 跨媒介预览 → 印刷检测 → 修正问题 → 导出印刷
```

## 常见问题

### Q1：导入图像后不显示？

**A**：请确认图像格式受支持（PNG、JPG、TIFF），并检查图像文件完整性。

### Q2：色差 ΔE 为 0 正常吗？

**A**：正常。ΔE 为 0 表示 RGB 和 CMYK 转换后再转回 RGB 的颜色与原始完全一致（理论值）。实际印刷中还会有其他因素影响。

### Q3：软打样预览与实际印刷有差异？

**A**：软打样是基于色彩转换的模拟，实际印刷结果会受纸张、油墨、打印机等因素影响。建议作为参考而非绝对标准。

### Q4：检测显示"分辨率不足"但图像很清晰？

**A**：屏幕上显示清晰不代表印刷精度足够。屏幕显示 72 DPI 即可，但印刷需要 300 DPI。建议使用高分辨率源文件。

### Q5：如何添加更多 ICC 配置文件？

**A**：目前版本支持预设的 ICC 配置列表。未来版本将支持用户加载自定义 ICC 文件。

### Q6：导出功能在哪里？

**A**：当前版本为开发完成阶段，导出功能需要后续版本实现。
