### 2.6 IPC 处理器

#### `dialog:openFile`

```ts
ipcMain.handle('dialog:openFile', async (_, options?) => {
  const defaultFilters = [{ name: 'Images', extensions: ['png','jpg','jpeg','tiff','tif','bmp'] }]
  const result = await dialog.showOpenDialog({
    properties: ['openFile'],
    filters: options?.filters || defaultFilters
  })
  if (result.canceled || !result.filePaths[0]) return null
  const buffer = await readFile(result.filePaths[0])
  return { filePath: result.filePaths[0], buffer: new Uint8Array(buffer) }
})
```

#### `dialog:saveFile`

- 根据 `options.extension` 选择保存过滤器:`png` / `jpeg` / `tiff`;
- 写入 `result.filePath`,返回 boolean。

#### `fs:readFile`(路径安全)

```ts
if (filePath.includes('..') || filePath.includes('~') || filePath.includes('$')) {
  throw new Error('Path traversal not allowed')
}
```

- 拒绝路径遍历;读取后返回 `Uint8Array`。

#### `project:load` / `project:save`

- 加载:`dialog.showOpenDialog` 过滤 `.pbp`,读取 UTF-8,`JSON.parse`;
- 保存:`dialog.showSaveDialog` 过滤 `.pbp`,`JSON.stringify(data, null, 2)` 写入。

#### `image:process`(占位)

```ts
ipcMain.handle('image:process', async (_, buffer, _options) => buffer)
```

- 当前实现为占位,直接回传 buffer;后续可在主进程注入更重的处理(目前由 Renderer Worker 完成)。

### 3.3 菜单订阅实现

```ts
onMenuAction: (callback) => {
  const listeners = MENU_ACTIONS.map((action) => {
    const channel = `menu:${action}`
    const handler = () => callback(action)
    ipcRenderer.on(channel, handler)
    return { channel, handler }
  })
  return () => {
    for (const { channel, handler } of listeners) {
      ipcRenderer.removeListener(channel, handler)
    }
  }
}
```

- 一次性注册所有菜单通道,返回统一解绑函数,避免泄漏。

## 5 工具层(`src/renderer/utils`)

### 5.1 color-convert.ts

#### 类型

```ts
type PaperType = 'coated' | 'uncoated' | 'newsprint'
interface RGB { r: number; g: number; b: number }
interface CMYK { c: number; m: number; y: number; k: number }
interface LAB { l: number; a: number; b: number }
```

#### 纸类型色域边界

```ts
const PAPER_GAMUT_BOUNDS = {
  coated:    { minL: 5,  maxL: 95, maxC: 150 },
  uncoated:  { minL: 8,  maxL: 92, maxC: 120 },
  newsprint: { minL: 10, maxL: 88, maxC: 85 }
}
const INK_LIMIT_FACTORS = { coated: 1.00, uncoated: 0.95, newsprint: 0.85 }
```

#### `clampToPaperGamut(lab, paperType)`

- 裁剪 L 到 [minL, maxL];
- 若 chroma 超 maxC,按比例缩放 a、b。

#### `rgbToCmyk(rgb, paperType = 'coated')`

控制流:

```
1. 边界:R=255,G=255,B=255 → CMYK(0,0,0,0)
2. 边界:R=0,G=0,B=0 → CMYK(0,0,0,100)
3. rgbToLab(rgb)
4. clampToPaperGamut(lab, paperType)
5. labToRgb(clamped)
6. K = 1 - max(R,G,B)/255
7. effectiveK = min(K, inkLimit)
8. C = (1 - R/255 - effectiveK) / (1 - effectiveK) * 100
9. M/Y 类似
10. clamp [0,100] 并四舍五入
```

#### `cmykToRgb(cmyk, paperType = 'coated')`

```
1. 边界:K=100 && C=M=Y=0 → RGB(0,0,0)
2. 边界:K=C=M=Y=0 → RGB(255,255,255)
3. 公式:R = 255*(1-C/100)*(1-K/100),G/M 同理,B/Y 同理
4. rgbToLab → clampToPaperGamut → labToRgb
```

#### `deltaE(rgb1, rgb2)`

- 使用 chroma-js 的 CIE2000 实现,入参转换为 hex。

#### `rgbToLab` / `labToRgb`

- 委托给 chroma-js。

#### `applyLightSource(r, g, b, source)`

| source | rMul | gMul | bMul |
| --- | --- | --- | --- |
| D50 | 1.05 | 1.02 | 0.95 |
| D65 | 0.98 | 1.00 | 1.02 |
| F | 0.95 | 1.05 | 0.98 |

- 输出为四舍五入后 clamp 至 [0, 255]。

#### `getSharpnessFactor(viewingDistance, resolution)`

```ts
distanceFactor = (250 / viewingDistance) ^ 1.4
resolutionFactor = (resolution / 300) ^ 0.9
return clamp(distanceFactor * resolutionFactor, 0.35, 1.7)
```

#### `analyzeColor(imageData, targetProfile)`

```
samples = sampleImagePixels(imageData, 5)   // 25 个采样点
尝试创建 sRGB → targetProfile 的 transform
若成功且 targetProfile 是 cmyk → useIccConversion = true

for sample in samples:
  originalColor = { r, g, b }
  若 useIccConversion:
    cmsDoTransform(transform, input, output, 1)
    convertedColor = { c, m, y, k } = output[0..3]

    convertedColor = rgbToCmyk(originalColor)
  backToRgb = cmykToRgb(convertedColor)
  dE = deltaE(originalColor, backToRgb)
  累加 totalDeltaE、maxDeltaE
  若 dE < 3 → inGamutCount++

deleteTransform(transform)

返回 {
  representativeColor = 平均 RGB
  convertedColor = rgbToCmyk(representativeColor)
  averageDeltaE, maxDeltaE
  isInGamut = averageDeltaE < 3
  inGamutPercentage = inGamutCount / sampleCount * 100
  sampleCount
}
```

### 6.3 print-checker.ts

#### 类型

```ts
type ProblemType = 'color-mode' | 'low-resolution' | 'missing-bleed' | 'fonts-not-embedded' | 'out-of-gamut'
type ProblemSeverity = 'error' | 'warning' | 'info'
type ProblemConfidence = 'high' | 'medium' | 'low'
type ProblemCategory = 'deterministic' | 'heuristic' | 'advisory'

interface PrintProblem {
  type, severity, category, confidence,
  title, description, location?, suggestedFix?
}
interface PrintCheckResult {
  problems: PrintProblem[]
  canPrint: boolean
  overallScore: number
  heuristicWarnings: number
  deterministicIssues: number
}
```

#### `checkPrintReadiness(imageData, options = {})`

```
options = { minResolution = 300, requiredBleed = 3, paperSize = 'A4', customWidthMm?, customHeightMm?, gamutThreshold = 3 }

1) estimatedDpi = min(imageW / (paperW / 25.4), imageH / (paperH / 25.4))
   若 estimatedDpi < minResolution → error 'low-resolution'
   若 estimatedDpi < minResolution * 1.2 → warning 'low-resolution'

2) 采样 outOfGamut:
   step = max(5, floor(max(W,H) / 100))
   统计 outOfGamutCount、sampleCount
   ratio = outOfGamutCount / sampleCount
   ratio > 10%  → error
   ratio > 1%   → warning
   else > 0     → info

3) checkBleed(imageData):
   calculateEdgeVariance(imageData, 30) > BLEED_VARIANCE_MIN(10)
   若 false → warning 'missing-bleed'

4) detectColorMode(imageData):
   若 grayRatio > 0.95 || avgSaturation < 0.15 → 'gray'
   否则 → 'rgb'
   'rgb' → warning 'color-mode'(advisory)
   'gray' → info 'color-mode'(advisory)

5) 统计:
   score = max(0, 100 - errorCount*30 - warningCount*10)
   canPrint = (errorCount === 0)
   deterministicIssues = count(problem.category == 'deterministic' && severity != 'info')
   heuristicWarnings = count(problem.category == 'heuristic' && severity != 'info')
```

#### 辅助函数

- `resolvePaperSize(options)`:处理 'custom' + 自定义尺寸;
- `calculateDpi(imageW, imageH, options)`:基于纸张 mm 推算 DPI;
- `isOutOfGamut(color, threshold = 3)`:CMYK 回算 RGB 后 ΔE;
- `sampleOutOfGamut(imageData, threshold)`:大步长采样;
- `checkBleed(imageData)`:边缘方差比较;
- `detectColorMode(imageData)`:灰度判定。

### 6.4 image-processor.ts

#### `simulatePrintPreview(imageData, options)`

```
saturationAdjust = paperType: newsprint 0.7 / uncoated 0.85 / coated 0.95
contrastAdjust   = paperType: newsprint 1.1 / uncoated 1.05 / coated 1.0
sharpnessFactor  = getSharpnessFactor(viewingDistance, resolution)
blurRadius       = max(0, round((1 - sharpnessFactor) * 3))

遍历像素:
  applyLightSource(r, g, b, lightSource)
  cmyk = rgbToCmyk(rgb, paperType)
  rgb' = cmykToRgb(cmyk, paperType)
  rgb' *= saturationAdjust
  contrast 调整:((rgb'/255 - 0.5) * contrastAdjust + 0.5) * 255
  clamp [0, 255]

若 blurRadius > 0:
  对中间结果做 box blur(邻域平均),提升模糊观感
否则:
  直接输出
```

### 6.5 ai-color-advisor.ts

#### `extractColorData(imageData)`

```
以 step=10 采样像素
累加 totalR/G/B、min/max
统计:saturatedPixels(>100 通道差)、skinTonePixels(R>95,G>40,B>20,R>G>B,|R-G|<35)
warmPixels(R > B+20)、coolPixels(B > R+20)

brightness = 0.299R + 0.587G + 0.114B
contrast = max(maxR-minR, maxG-minG, maxB-minB)
colorfulness = saturatedPixels / (pixelCount / step)

contrast:    >150 → 'high'; >80 → 'medium'; else → 'low'
saturation:  colorfulness>0.35 → 'high'; >0.15 → 'medium'; else → 'low'
temperature: warm > 1.5*cool → 'warm'; cool > 1.5*warm → 'cool'; else 'neutral'
hasSkinTones: skinTonePixels > 30
sceneType:   colorfulness>0.4 → 'graphic'; >0.15 → 'mixed'; skinTonePixels>50 → 'photo'; else 'unknown'
```

## 14 安全设计细化

### 14.1 进程隔离

- `contextIsolation: true`:Preload 与 Renderer 隔离;
- `nodeIntegration: false`:Renderer 不直接访问 Node API。

### 14.2 API 白名单

Preload 仅暴露以下方法:

```
openFile / saveFile / readFile / processImage / loadProject / saveProject / onMenuAction
```

### 14.3 路径校验

- 主进程 `fs:readFile` 拒绝包含 `..` `~` `$` 的路径;
- Preload `readFile` 错误重抛,Renderer 层据此 toast 错误。

### 14.4 依赖审计

- 仅使用 MIT/BSD/Apache 协议依赖(详见可行性报告)。

## 15 性能细节

### 15.1 采样策略

- 颜色分析:固定 5×5 = 25 采样点;
- 印前色域采样:步长 `max(5, floor(max(W, H) / 100))`,默认 ~100×100;
- 出血检测:`calculateEdgeVariance(imageData, 30)`,仅取 30px 边缘带。

### 15.2 Web Worker

- 重计算放在 Worker,主线程保持响应;
- 失败时降级到主线程同步处理并提示用户。

### 15.3 持久化裁剪

- `partialize` 排除大体量与可重算字段。

### 15.4 渲染优化

- Zustand selector 精确订阅,避免无关组件重渲染;
- Ant Design + CSS 变量,主题切换无重渲染抖动。
